# SP4M
SP4M: Hundred Laugh is a simple message spamming tool.

# Usage
For Debian/Arch Linux:
* git clone https://github.com/xfr0g/sp4m.git
* cd sp4m
* chmod +x installer.sh
* ./installer.sh
* python3 sp4m.py

For Termux:
* pkg update && upgrade
* pkg install python3 git
* git clone https://github.com/xfr0g/sp4m.git
* cd sp4m
* chmod +x installer.sh
* ./installer.sh
* python3 sp4m.py
